À propos de forfaits

App-forfait est est application web qui offre des services de forfaits de toutes sortes pour satisfaire le besoin de sa clientèle.
Elle est developpée en framework Angular open-source utilisant un modèle de programmation orienté composant pour structurer le code et faciliter d'une application monopage coté client à l'aide de TypeScript et de HTML et aussi un ensemble des composants d'Angular materials qui est une librairie d'interface utilisateur basée sur Material Design de Google, qui fournit des composants prédéfinis pour donner à l'application une apparence professionnelle et moderne.
Angular et Material Angular nous permettent de créer des applications web performantes et élégantes en utilisants des composants réutilisables. Ca nous permet d'utiliser les fonctionnalités avancées d'angular comme la liaison de données, la validation de formulaire, et tant d'autres encore. Cette application a été developpée dans le cadre d'un projet de AEC en développement web Front-End - Cégep de Trois-Rivières, proposée par Carine Croteau.
